# Regras do Projeto

## Commits

- Usar **Conventional Commits** em português
- Formatos: `feat:`, `fix:`, `docs:`, `style:`, `refactor:`, `test:`, `chore:`, `perf:`, `ci:`, `build:`
- Mensagens de commit sempre em **português**
- **Não** incluir `Co-Authored-By`
- **Sempre** apresentar a mensagem de commit ao usuário para confirmação antes de executar o `git commit`
